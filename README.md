# Raven Ring

## Overview
A fantasy-themed, retro-game-styled website where House Stark sends sealed,
ciphered scrolls to Jon Snow, a magical hobbit guarding the Wall. Visitors
can crack open a sample scroll, or use the built-in Scriptorium to encode
and decode messages of their own.

## Features
Every scroll is sealed with the same spell: for each sentence, count the
letters in its first word — that number is the shift. Every letter in that
sentence is then moved forward through the alphabet by that amount to seal
it, or backward by that same amount to unseal it.

## Technologies Used
Built with plain HTML/CSS/JS — no framework, no build tooling. Works in all
current evergreen browsers (Chrome, Firefox, Safari, Edge). Uses:
localStorage, the Clipboard API (for "Copy result"), and the File API (for
.txt uploads) — all standard and widely supported.
